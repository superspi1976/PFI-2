# Proyecto Final Integrador 


Aplicacion en Python que para gestion de inventario, la app esta diseñada para:

    1. Agregar 
    2. Mostrar 
    3. Actualizar Cantidad de 
    4. Eliminar 
    5. Buscar 
    6. Reporte de Bajo Stock
    
A tal efecto se utilizo una base de datos SQLite para almacenar
los datos de los productos para interactuar con la misma desde la terminal


El proyecto tiene 2 archivos principales:

- app.py: Controla el flujo principal de la aplicación y muestra el menú principal. Contiene funciones auxiliares para mostrar menúes y solicitar datos al usuario
- funciones_db.py: Gestiona las operaciones con la base de datos SQLite

Y un 3 archivo que es la base de datos generada al efectuar las pruebas
